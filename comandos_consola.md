git --version → verifica si Git está instalado

git config --global user.name "Nombre" → configura tu nombre en Git

git config --global user.email "email" → configura tu correo en Git

mkdir nombre carpeta → crea una carpeta

cd nombre carpeta → entra a una carpeta

git init → inicializa un repositorio Git

git status → muestra el estado de los archivos

git add README.md → añade un archivo al área de preparación

git add . → añade todos los archivos al área de preparación

git commit -m "mensaje" → guarda un punto de control (commit)

git log --oneline → muestra el historial de commit
